Description

Scope is a range withing a program for which an item is valid. By item we usually mean variables.

Analogue of scope in Russian: область видимости

.

Kinds of scopes

We distinguish two types of scopes:

  1. Global scope. Variables defined outside functions, classes and structures live in global scope and called global variables
  2. Local scope. Variables defined in functions live in local scope, allocated on stack and called local variables.
Usually global variables live during the entire program. On the opposite side local variables live only inside the pair of curly brackets where it's defined. When a local variable goes out of scope its value will be dropped.

What is a scope in programming languages?

In almost every programming language scope is denoted by the pair of curly brackets {}.

Note: only in Pascal programming language scope is denoted by two key words: begin and end
