Definition

In computer science, an expression is a syntactic construction that can be evaluated to it's value. More strictly, expression - is a combination of one or more literals, constants, variables, functions that programming language computes to a certain value.

Examples

For instance, 3+8/4 - expression, because it can be computed to a result 5. Functions calls: get_sum(3,5) is also an expression - it returns a value 8, of course if it have been defined properly.

Pitfalls

Usually beginner programmers don't distinguish terms expression and statement. You should always remember, that expression is characterized by it's result value. Whereas statements in some programming langugages (for example, in C, C++> and Rust) always followed by a semicolon ; - the sign of end of code line.
