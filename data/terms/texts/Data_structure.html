Definition

A data structure is a particular way of organizing data in a computer programs so that it can be used in effective and convenient manner.

Analogue of data structure in Russian: структура данных.

Analogue of collection in Russian: коллекция

Usually data structures supposed to create a user-defined types for representing real-world entities. Say we're developing a program for maintaining taxi car park. In such project we can create entity car and store in that data structure all related information:

  • Manufacturer's brand
  • Fuel tank volume
  • Current mileage
  • Number of seats

To recup the above: programmers creates data structures that suit every their needs for every task. Nevertheless the shouldn't create every data structure from scratch. Fantastic amount of any available data structure was created during all time. So programmer can just use the results of his colleages or open-source community. Besides every programming language standart library has the most appropriated implementation of widespread data structures for a particular tasks.

Some data structures have another name: collection, they are used for storing a values in form of:

  • Sequences. The most pervasive sequences in programming languages are character sequences or simply strings.
  • Graphs - is a native approach for storing routes (even whole road maps) or networks (computer networks, social networks). Every net is represented by two sets: set of vertexes(points) and set of edges(routes). Moreover graphs let programmer to develop sophisticated and comprehensive algorithms: finding possible ways(usually shortest or less expensive meaning costs)

Common data structures in programming languages

  • Array (static length contiguous sequence of elements)
  • Vector (dynamic length contiguous sequence of elements that can grow and shrink on demand)
  • Linked list (double-linked list)
  • Queue (/kjuː/ pronounces like letter q)
  • Stack
  • Binary tree (binary search tree, kind of graph)
  • Heap
  • Hashing
  • Matrix (table of values)

In conclusion we have to notice that data types are data structures too. Sometimes they are called primitive data structures because they only gives a meaning to memory blob.
