Definition

In computer science and computer programming, a data type or simply type is an attribute of data which tells the compiler or interpreter how the programmer intends to use the data.

As you may know memory can contain only binary representations of data. So it's hard to work with such raw data. Thus computer scientists has invented data types to simplify programs development. Now developers are intended to use data of a particular type instead of a raw memory blobs without any meaning. Only its creator bringing means to it: wheter it a number, image, driver, library, part of a game data, etc.

Analogue of data type in Russian: тип данных.

Common data types

In programming languages the most common data types are:

Numbers

  • Integer - represents the signed 32 bit number without fractional part. For the sake of brevity in programming languages are used it's shortened forms: int, i32. Variables of integer type can deal with numbers from [-2^32..2^32-1] range. P.S. this range includes zero so we need to subtract one from the rightest border of the range.
    • Short - a shortened version of an integer type. It consist of only 16 bits. Variables of short type can deal with numbers from [-2^16..2^16-1] range.
  • Long - opposite to a previous type - longed version of integer type. It consist of 64 bits and can handle numbers in a range [-2^64..2^64-1].

It worth noting that in some programming languages there're unsiged number types. It means that variables can't hold negative values and can only work in ranges [0..2^16], [0..2^32], [0..2^64] for types unsigend short, unsigned int and unsigned long respectively.

Floating-point numbers

  • Float - 32 bit number with a floating point
  • Double - 64 bit number with a floating point

Characters and character sequences

  • Char. In old programming languages such as C and C++ variable char type can hold one symbol from a ASCII table (american code page that contains all latin-alphabet symbols, digits, and some other symbols on your keyboard). In modern languages, for instance in Rust chars can store symbol of UTF-8 code page (international code page that can hold symbols of every language on Earth)
  • String - a dynamic sequence of char elements.

Logic types

Logic type is single and presented by boolean type (in languages simply bool). Usually it takes 1 byte though it requeres only 1 bit because boolean variables can hold only two possible values: true and false.
