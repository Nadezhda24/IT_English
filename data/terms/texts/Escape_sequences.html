Definition

Escape sequences let us to embed characters in strings that cannot easily be typed on a keyboard. For instance, if you want to embed a new line symbol into some string in your code, you can't simply press Enter button, because it's incorrectly for programming languages to separate expressions by accidental new line symbols. To do this properly, you have to use special symbol: "\\n" which refer to symbol with code 10 in ASCII table.

It worth noting that escape sequences work only with non-printable symbols: tabulations (\\t, табуляция), line feeds (\\n, перенос строки), carriage return (\\r, возврат каретки), beep (\\a, звуковой сигнал).

Analogue of escape-sequence in Russian: управляющая последовательность.

Representing Unicode symbols

In some programming languages, in Python for instance, you can code string Unicode literals with help of escape sequences. To do this you need to put "\\" symbol and the hexadecimal code of symbol. For example: "\xF578".
