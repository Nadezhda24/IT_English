Definition

Generally, slicing is an operation that extracts a subset of elements from a sequence.

Analogue of slice in Russian: срез, отрезок.

Definition pitfalls

The difficult is some programming languages defines slice in their own way. For instance, in Rust programming language slice is defined as a reference to a contiguous sequence of elements of a collection, i.e. here you can define a reference to a part of sequence rather than to a whole sequence.

Whereas in Python programming language slice lets you to construct a new sequence from the part of an existing one.

Syntax of slices in different languages

In Rust you can create a slice in following way: &s[0..5], assuming that s is a string. This sentence takes 5 symbols - from 0th up to 4th positions.

In Python you create references by typing s[0:4], assuming that s is a sequence (list, tuple, string). This sentence takes 4 symbols from - 0 up to 3rd.

What languages let you to use slices?

Slice should be considered as a complex construction. Not every programming language provides ability to get slices. For instance, Rust and Python - relatively new programming languages do, and such languages as C don's.
