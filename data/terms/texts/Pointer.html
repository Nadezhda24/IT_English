Definition

A pointer is a variable that stores a memory address.

Analogue of pointer in Russian: указатель.

Dynamic objects

Usually pointers are used for storing an address of memory allocated for dynamic object(s) which are created on a Heap. It's hard to predict carefully how much memory your program will consume, because this value depends on many other values such as program working duration, amount of input data and so on.

As a result developers get an opportunity to allocate as much memory as needed. The problem is this memory has to be cleared manually in some languages (C and C++). This makes using pointers one of many advanced topics in programming.

Smart pointer

Due to the previous difficulties developers invented so-called smart pointers. This is an implementation of RAII(Resource Acquisition Is Initialization) programming paradigm. As for pointers this principle means that pointer will be cleared by it's owner object automatically after going out of scope.

For instance, in C++ programming language smart pointer types are following:

  • Unique pointer (unique_ptr). Fully owns the memory it is given and can't share it with other objects.
  • Shared pointer (shared_ptr). Apparently allows sharing resources between other shared pointers.
  • Weak pointer (weak_ptr)

Pointers in modern programming languages

By virtue of complexity of using pointers in many modern programming languages developer have no possibility to use it. For example in Python there aren't any pointers in their pure meaning. Variables are already pointers and these terms are merged for simplifying.
