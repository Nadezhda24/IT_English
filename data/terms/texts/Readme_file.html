Definition

README is a text file for primary acquaintance with a particular project. For instance, every popular open source project has a README file in it's repository at GitHub. Rarely README file even can contain an instruction for installation a program, but this is useful only for it-specialists.

There isn't any appropriate analogue of readme in Russian.

How to create a README file for a project?

You don't need any special skills for creating a README file for a project - there is nothing to it - just follow this list and add appropriate info:

  • The name of a project
  • A brief description of project's purposes
  • Usage examples in case you create a library
  • Installation notes in case you create an application

Format of README files

For a sake of readability README files uses some markup language, the most popular one is - markdown. This language is used in GitHub repositories.
