Definition

Boolean (bool) - is one of fundamental data types in programming, it represents a logical type. Usually in most programming languages it takes 1 byte, but theoretically it requires only 1 bit, because there're only 2 possible values: true or false.

Analogue of boolean in Russian: булевский тип, логический тип

Why is not this type called logical?

We have a reasonable answer: the name of boolean type was given in honor of the English mathematician and logician George Boole. He is the first who used the logical types in one of his articles.

Usage

Usually booleans are used for storing the results of computing of conditions. Moreover they can be used as flags in system programming, but most likely every experienced programmer will choose integers for this.

Exceptions

In some old versions of C language there isn't a bool type. Instead of it language implicitly treated every non-zero integers as true values and zeroes as false. Nevertheless C-maintainers added a library for boolean type in future versions.
