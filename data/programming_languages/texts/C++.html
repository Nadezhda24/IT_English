Creators

The C++ language was developed by Danish computer scientist Bjarne Stroustrup. Today C++ has a working group which continues to add new functionality and maintain the language.

Date of the first release

The C++ first release was in 1985. In those days C++ has another name - C with Classes. Despite of this C++ was standartised only in 1998. After that the versions of C++ get the following names: C++03, C++11, ... C++20. The last two digits are the last digits of the year in which a version has been introduced.

Brief description

C++ can be pronunced like "si: plʌs plʌs"

C++ is a general purpose programming language and widely used now a days for competitive programming. It has imperative, object-oriented and generic programming features. C++ is runs on lots of platform like Windows, Linux, Unix, Mac etc. C++ is intended for high-loaded systems creation because this is almost the only programming language which gives a programmer the full power of system control:

  1. Manual memory control
  2. Bitwise operations
  3. Low-level programming constructions
  4. Extensive multithreading toolkit

One will say that C++ is almost dead programming language because it was created in 1985. This is completely wrong idea. A new C++ version releases every 3 years. Every time it brings a lot of new features to the wide programming community. It's fantastically if you know that initially it was a small extention above C programming language.

A lot of IT-companies uses it in their systems. At the end of this article you will see how many popular programs were written via C++.

Purposes

As I have already mentioned C++ is a general-purpose programming language. It means that programmer can create almost every program via C++. But initially C++ was designed with an orientation toward systems programming and embedded, resource-constrained software and large systems, with performance, efficiency, and flexibility of use. Certainly C++ has also been found useful in many other contexts including:

  1. Desktop applications
  2. Video games
  3. E-commerce servers
  4. Databases
  5. Performance critical applications
