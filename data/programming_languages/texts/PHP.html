PHP - a dynamic interpreted scrypting language for building interactive websites on the server. it was created in 1994 by Rasmus Lerdorf to manage his Personal Home Page. But today developers community describe PHP like PHP Hypertext Preprocessor (recursive acronym).

PHP has a special place in the history of the Internet because it predates JavaScript and it was the first language to be embedded directly in HTML. Today it's one of the most popular languages for back-end web development. Inspite of being interpreted language, Facebook built a custom compiler to convert it directly to machine code.
