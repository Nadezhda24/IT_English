Description

C ([si:] - is pronounced like the letter c) - is a general-purpose, procedural programming language. It has been developed by Dennis Ritchie between 1972 and 1973 at Bell Labs (american company). Basically it was used for creating Unix utility-tools (Internally, Unix fully written in C, with the rarely exception of kernle, maybe).

It supports structure programming, recursion and variable scopes. By design this language has static type system which means that compiler have to know a type of every using variable at compile-time.

Professionals say that C is a father of almost every modern programming languages. It's customary to say that some languages are c-style. It means that syntax of some language is very familiar with syntax of C programming language. For instance, C-style languages are: C++, C#, Java (more precisely is came from C++), Rust, Ruby, Python, Perl, PHP and several more.

For this reason C language became an academic language - it's used in teaching of young developers at universities and even at some schools.

Use of C programming language in our modern world

Despite of it's age C is widely used in modern world. The newer version of this language is called C17 (version of 2018 year). Foremost this language is used in microcontroller programming. This is due to the fact that C has low-level instruments for: manual memory management (it increases speed of program), bitwise operations, which are basic interface for working with I/O pins of microcontroller.
