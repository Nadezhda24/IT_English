Definition

Programmer(sometimes called a software developer, software engineer or even coder) is a person who creates computer software. More precisely software engineer is a specialist in several areas of computer science.

Analogue of programmer in Russian: программист, разработчик программного обеспечения

Programmer's level

In real vacancies programmers are divided into several positions:

  • Junior software developer. This is a name of entry-level programmers who need to be guided by their elder teammates in many technical questions.
  • Middle software developer. This is a name of intermediate experienced programmers who can perform medium-sized tasks alone.
  • Senior software developer. This is a name of software developers with many years of experience in solving business problems with programs.
  • Team-lead software developer. Usually in componies programmers don't work alone because it's hard to write entire program by oneself. Instead of this programmers work in teams. There is a leader of that group - team-lead. He is in charge of dividing of tasks between participants of team and of new-hire programs.
It's obvious that programmers salary depends on his level and experience.

History

British countess and mathematician Ada Lovelace is often considered to be the first computer programmer, as she was the first to publish part of a program (specifically an algorithm) intended for implementation on Charles Babbage's analytical engine in 1842. The algorithm was used to calculate Bernoulli numbers. Because Babbage's machine was never completed as a functioning standard in Lovelace's time, she never had the opportunity to see the algorithm in action.

Later in 1979 Ada programming language was created and named after the first programmer - Ada Lovelace.
